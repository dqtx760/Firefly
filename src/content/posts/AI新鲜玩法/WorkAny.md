**WorkAny** 是一款基于 Claude 3.5 Sonnet 模型构建的智能工作助手平台。它不仅能与你自然对话，更拥有强大的文件处理、代码编写、文档创建等实用能力。简单来说，它就像给 AI 装上了"双手"——不仅能思考，还能真正帮你**动手完成工作**。

![](https://gitee.com/da-qiang-classmate/typora/raw/master/image/workany_coding_1280x720.webp)


## ✨ 核心功能一览

### 💻 代码开发
编写、调试、优化代码，支持 JavaScript、Python、TypeScript、Java、Go 等多种编程语言。

> **场景**：快速生成 API 接口、编写自动化脚本、调试代码错误

### 📝 文档处理
- 创建 Word、Excel、PDF 文档
- Markdown 文档编辑与转换
- 文档格式转换与排版

> **场景**：自动生成报告、批量处理 Excel 数据、撰写技术文档

### 🔍 智能搜索
实时网络搜索、资料查询、知识问答，快速获取你需要的信息。

> **场景**：查找资料、验证信息、学习新知识

### 🌐 网页内容抓取
一键获取网页内容，提取关键信息，支持批量数据采集。

> **场景**：竞品分析、数据收集、内容整理

### 📊 数据处理
- Excel 数据分析与图表生成
- CSV/JSON 数据转换
- 数据清洗与格式化

> **场景**：销售数据分析、客户信息整理、报表生成

### 🎨 AI 配图生成
支持调用本地 AI 绘图技能，一键生成高质量配图，支持自定义尺寸和风格。

> **场景**：公众号配图、文章封面、演示文稿插图

**本地 AI 绘图能力**：
- 支持通义千问 AI 绘图模型
- 自定义图片尺寸（如 16:6 封面、1280x720 配图等）
- 中文描述生成，简单易用
- 批量生成，提升效率

### 📱 公众号助手
Markdown 自动转换为公众号格式，一键上传草稿箱，支持多种精美主题。

> **场景**：快速排版公众号文章、批量内容生成

### 🔌 本地技能扩展
支持调用你本地配置的自定义技能，无限扩展功能边界。

> **场景**：调用本地 AI 绘图生成配图、使用自定义 API 服务

**技能系统亮点**：
- 灵活配置本地技能目录
- 支持自定义 API 集成
- 一键调用，无缝使用
- 本地优先，数据安全

### ⚙️ 自动化工作流创建
一键创建专属工作流，将重复性任务自动化，让你发送一条指令就能完成一系列复杂操作。

> **场景**：文章创作工作流、数据处理流水线、批量任务自动化

**工作流系统亮点**：
- 智能识别任务需求，自动生成工作流脚本
- 支持文章生成+配图制作一体化完成
- 简单指令触发复杂流程，一键执行
- 完全可定制，适配各种业务场景
- 自动保存结果，文件分类管理

**实际应用示例**：
只需发送一条指令「/write 人工智能的未来发展」，WorkAny 将自动完成：
- ✍️ 撰写完整文章内容（无分割线、无表格）
- 🖼️ 生成封面图（1920x720，16:6 比例）
- 🎨 生成 3-6 张配图（1280x720，16:9 比例）
- 📁 自动分类保存到指定目录

一次配置，永久受益，让重复工作彻底自动化！
